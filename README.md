#### Server

- JavaScript
- Node.js, Express

## Installation and Usage

1. Fork this repository.
2. Clone your fork.

```bash
git clone https://github.com/your_username/repository_name.git
```

3. Navigate to the project directory.

```bash
cd repository_name
```

4. Install the dependencies.

```bash
npm run install:all
```

5. Start the project.

```bash
npm start
```

6. You can view the project at

```txt
http://localhost:5173/
```
